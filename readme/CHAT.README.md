## 채팅에 관한 실제 화면과 간단한 요구사항이에요.

### 실시간 대화, 채팅 목록
- 최근 매세지와 **분 단위**로 차이가 나지 않으면 전송한 시간은 표시 하지 않아요.
- 간병 신청을 **거절**하거나 **수락**했을 경우에는 **자동**으로 매세지가 전송이 돼요.  
- 간병인의 응답 상태에 따라 **방 상태**가 변경돼요.  
- 채팅 목록에는 **실시간**으로 미확인 매세지의 갯수가 표시되고 읽으면 사라져요.
- 방 목록에는 가장 최근의 매세지가 있는 방이 **항상 상단**으로 이동해요.

### 간병 신청
- **간병인끼리**는 간병 신청을 할 수 없어요.
- 보호자가 간병인에게 **간병 신청**을 하면 자동으로 채팅방이 개설돼요.
- 개설되면서 보호자의 **환자분 내용과 신청서**가 간병인에게 채팅방을 통해 전달이 돼요.
- 만약 이미 대화 내역이 있는 경우 **해당 채팅방으로 이동**하게 돼요.
- 간병인은 이미 응답을 했다면 **재응답**은 불가능해요.
